# yh-mall 一个典型移动端vue商城项目

## 项目起步

```javscript
npm install
```

### 编译并加载项目

```javscript
npm run serve
```

### 编译上线资源

```javscript
npm run build
```

### Lints and fixes files

```javscript
npm run lint
```
